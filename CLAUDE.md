# Project Instructions

## Documentation

All project markdown documents live under `.docs/`:

| Directory | Purpose |
|-----------|---------|
| `.docs/ideas/` | Specs, concepts, brainstorming documents |
| `.docs/plans/` | Implementation plans |

**Convention:** Name files with a date prefix: `YYYY-MM-DD-description.md`

## Tech Stack

- Vanilla JavaScript (ES modules) with Vite
- Three.js for 3D rendering
- No framework, no TypeScript

## Development

```bash
npm run dev     # Start dev server
npm run build   # Production build
```
